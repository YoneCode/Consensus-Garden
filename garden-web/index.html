FAQ
Questions
What is Consensus Garden? +
A 2,000-supply, fully on-chain NFT collection on GenLayer. Each token is a living tree forged from a builder's GitHub identity and grown over real time — explorable in a 3D world.
Why is this only possible on GenLayer? +
GenLayer contracts can fetch the live web and run AI, then validators reach consensus on the result — trustlessly, with no oracle. That's how the tree reads your GitHub, knows the real-world time, and shows live prices. A normal EVM chain cannot do any of this on-chain.
Is it really free? +
Yes — gasless. A sponsor wallet mints your tree for you, so there's no fee and no wallet popup. One tree per wallet.
Who can mint, and when? +
Open to everyone right now — GenLayer builders, the community, and the public. One tree per wallet, free and gasless, until all 2,000 are planted.
Does the tree actually grow? +
Yes. Growth is computed from GenLayer's on-chain clock — seedling within an hour, sapling in a day, mature in about a month, then it keeps slowly aging. It outlives the session.
Is the art AI-generated? +
No. The art is a deterministic generative algorithm rendered on-chain — no diffusion, no “AI slop”. AI is used only to judge your identity (which species you get), never to draw the image.
Where do I see my NFT? +
In the 3D garden, in a dedicated spinnable 3D viewer, and on the GenLayer explorer — the same tree from the same on-chain DNA.
